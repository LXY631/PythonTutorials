# 函数

## print 函数

- 默认输出并换行: `print("string")`
- 指定输出结束符: `print("string", end="__")`

## 自定义函数

``` PYTHON
def func_name():
  ...
  return None
```

- 形式参数(形参)
- 传递参数/实际参数(传参/实参)
- 返回值
- 函数嵌套调用，必须顺序正确

- 位置参数：必须传值，且有几个参数就传几个传值
- 默认参数：没有传值，使用默认参数

- 形参：
  - `*args`: 元祖
  - `**kargs`: 字典

``` Python
func(1,2,3, a='a',b='b',c='c')
args = (1,2,3)
kargs = {a:'a',b:'b',c:'c'}

li = [1,2,3]
func(*li) # 实参角度，序列按照顺序打散

def func(*args): # 形参角度，给变量加上*，就是组合所有传来的值
'''
#     这个函数实现了什么功能
#     参数1：
#     参数2：
#     :return: 是字符串或者列表的长度
#     '''
```

- 形参：
  - 位置参数 ： 必须传
  - *args ：可以接收任意多个位置参数
  - 默认参数 ： 可以不传
  - **kwargs ： 可以接收多个关键字参数

- 实参：按照位置传参，按照关键字传参
